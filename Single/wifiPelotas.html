WiFi Pelotas

WiFi Pelotas comes to help citizens of Pelotas (Brazil) to find WiFi signals around the city. In a reality where 3G and 4G signals fail often, this applications aims to provide a way to find a faster networks for those moments of need.

The application is powered by inputs from the own community in a public Google Spreadsheet document, which updates the pins in the map automatically.

Technologies I used

  • NodeJS
  • Google Maps API
  • Google Spreedsheats API
  • Google Drive - Spreadsheets (as a database)
