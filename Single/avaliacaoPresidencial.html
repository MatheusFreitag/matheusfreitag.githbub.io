Avaliação Presidencial

Brazil faced a militar dictatorship from 1964 to 1985. After this period the country entered what would later be called "Redemocratization", and since then 6 presidents governed the country.

This dashboard (in PT-BR) shows how public opinion evolved during each government, and how many brazilians(in %) rated each government as "Excellent/Good", "Ok" and "Bad/Terrible".

It is interesting to see how each president was portrayed in public opinion through time.

Technologies I used

  • HTML
  • CSS
  • ChartJS
  • VueJS
  • Pandas
  • Matplotlib
  • Numpy
